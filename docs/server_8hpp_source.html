7  * The Server class is an abstract base class that provides an interface for
8  * creating servers. The HttpServer class inherits from the Server class and
9  * implements an HTTP server that can receive and send HTTP requests and
10  * responses.
11  *
12  * The HttpServer class can be configured with a ServerConfig object that
13  * specifies the IP address and port number on which to listen for incoming
14  * connections.
15  *
16  * The HttpServer class currently supports only the GET method for serving
17  * static files. It can serve a default HTML file if the root URI is requested,
18  * or it can return a 404 Not Found error for any other URI.
19  *
20  * To use the HttpServer class, you must create an instance of it with a
21  * ServerConfig object, call the start() method to begin listening for
22  * connections, and then call the receiveRequest() method in a loop to receive
23  * incoming HTTP requests. Once you have received a request, you can use the
24  * handleRequest() method to generate a response, and then send the response
25  * with the sendResponse() method.
26  *
