10  * events. The KqueueEventListener and EpollEventListener classes inherit from
11  * the EventListener class and implement the event handling functionality using
12  * the kqueue and epoll system calls, respectively.
13  *
14  * To use the EventListener classes, you must create an instance of one of the
15  * derived classes, registerEvent the sockets you want to monitor using the
16  * registerEventSocket() method, and then call the start() method to begin
17  * monitoring events on the registerEvented sockets. Once you have received an
18  * event, you can use the handleEvent() method to process the event and take
19  * appropriate action, such as reading or writing data to the socket.
20  *
21  * @note The KqueueEventListener class is used on MacOS, and the
