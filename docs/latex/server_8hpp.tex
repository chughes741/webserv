\doxysubsection{Detailed Description}
Declares classes for creating web servers that can handle HTTP requests. 

This file contains the declaration of the \mbox{\hyperlink{classServer}{Server}} and \mbox{\hyperlink{classHttpServer}{Http\+Server}} classes. The \mbox{\hyperlink{classServer}{Server}} class is an abstract base class that provides an interface for creating servers. The \mbox{\hyperlink{classHttpServer}{Http\+Server}} class inherits from the \mbox{\hyperlink{classServer}{Server}} class and implements an HTTP server that can receive and send HTTP requests and responses.

The \mbox{\hyperlink{classHttpServer}{Http\+Server}} class can be configured with a \mbox{\hyperlink{structServerConfig}{Server\+Config}} object that specifies the IP address and port number on which to listen for incoming connections.

The \mbox{\hyperlink{classHttpServer}{Http\+Server}} class currently supports only the GET method for serving static files. It can serve a default HTML file if the root URI is requested, or it can return a 404 Not Found error for any other URI.

To use the \mbox{\hyperlink{classHttpServer}{Http\+Server}} class, you must create an instance of it with a \mbox{\hyperlink{structServerConfig}{Server\+Config}} object, call the start() method to begin listening for connections, and then call the receive\+Request() method in a loop to receive incoming HTTP requests. Once you have received a request, you can use the handle\+Request() method to generate a response, and then send the response with the send\+Response() method.
