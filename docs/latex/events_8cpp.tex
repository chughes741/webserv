\doxysubsection{Detailed Description}
Defines classes for handling events. 

This file contains the implementation of the \mbox{\hyperlink{classEventLoop}{Event\+Loop}}, Kqueue\+Event\+Loop, and Epoll\+Event\+Loop classes. The \mbox{\hyperlink{classEventLoop}{Event\+Loop}} class is an abstract base class that provides an interface for creating event loops. The Kqueue\+Event\+Loop class inherits from the \mbox{\hyperlink{classEventLoop}{Event\+Loop}} class and implements an event loop that uses the kqueue system call. The Epoll\+Event\+Loop class inherits from the \mbox{\hyperlink{classEventLoop}{Event\+Loop}} class and implements an event loop that uses the epoll system call.

\begin{DoxyNote}{Note}
The Kqueue\+Event\+Loop class is only available on Mac\+OS and the Epoll\+Event\+Loop class is only available on Linux.

This code is for educational purposes only and should not be used in production environments without extensive testing and modification.
\end{DoxyNote}
